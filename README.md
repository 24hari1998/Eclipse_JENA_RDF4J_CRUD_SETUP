# Eclipse_JENA_RDF4J_CRUD_SETUP
Eclipse setup for testing storage etc for official JENA and RDF4J

Instructions for Build:
1) This is a maven project. You can clone and import the project into your eclipse workspace. Build will automatically install necessary dependencies from pom.xml
2) Please unzip the file under resources for the project to work.

Supports:
1) Insert operation for both RDF4J and JENA.
